## Github Desktop汉化包
#### 支持 2.9.4版本的 Github Desktop
下载app.js和renderer.js文件

或者直接下载app.rar

打开Github Desktop安装位置 进入当前目录

​	``` ...\GitHubDesktop\app-2.9.4\resources\app ```

把 app.js和renderer.js 放到目录下，重启github客户端
